# Personas de usuario

Estado del documento: Vigente  
Ultima actualizacion: 2026-05-10

## 1. Trader principiante/intermedio

- Perfil: Opera con reglas parciales y poca revision.
- Problema: Repite errores sin medirlos.
- Necesidad: Journal simple, metricas y retos.
- Ayuda: Estructura de registro y feedback.
- Features: Journal, Challenges, AI Coach basico.

## 2. Trader activo/pro

- Perfil: Opera con frecuencia y busca consistencia.
- Problema: Necesita detectar patrones de perdida.
- Necesidad: Analytics, Mistake Tax, readiness.
- Ayuda: Reportes y retrospectiva semanal.
- Features: PRO, analytics, playbooks.

## 3. Academia de trading

- Perfil: Forma grupos de alumnos.
- Problema: Difícil medir disciplina grupal.
- Necesidad: Multiusuario y retos.
- Ayuda: Metricas grupales y panel admin futuro.
- Features: TEAM / ACADEMY.

## 4. Mentor o coach

- Perfil: Acompana traders.
- Problema: Feedback disperso.
- Necesidad: Evidencia, journal y reportes.
- Ayuda: Revision estructurada por datos.
- Features: Journal, AI Coach, reports.

## 5. Usuario tecnico/analista

- Perfil: Quiere datos y modelos de comportamiento.
- Problema: Falta de trazabilidad.
- Necesidad: Export, metricas, API futura.
- Ayuda: Data model y analytics.
- Features: Analytics, observability, future API.
