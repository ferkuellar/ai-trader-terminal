# ADR-0002: Landing first strategy

Estado del documento: Aceptado  
Ultima actualizacion: 2026-05-10

## Contexto

El producto necesita validar mensaje, posicionamiento, pricing y confianza antes de construir la terminal completa.

## Decision

Construir primero la landing publica.

## Razon

- Validar producto.
- Comunicar valor.
- Preparar presencia publica.
- Capturar leads en fase futura.
- Evitar construir backend antes de confirmar direccion.

## Consecuencias

- Avance rapido en percepcion de producto.
- Backend queda planeado.
- La documentacion debe distinguir mock vs real.
