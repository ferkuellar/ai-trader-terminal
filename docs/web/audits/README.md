Las auditorias documentan contexto, decisiones, validacion y riesgos por fase.

## Estructura obligatoria

1. Auditoria inicial
2. Problema detectado
3. Plan tecnico
4. Archivos creados
5. Archivos modificados
6. Implementacion
7. Validacion
8. Riesgos
9. Auditoria final
10. Enunciado del commit
